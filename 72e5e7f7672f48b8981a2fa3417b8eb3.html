List UI Controls

Lists allow you to configure various sorts of lists in your prototype page.

In Build, we distinguish between tables and lists. Both usually contain homogeneous data, but lists generally have rather basic data, whereas the data in tables tends to be more complex.

Table

Contains a set of line items, and is responsive to the display format. Similar to List controls, but data in a Table is more complex. Depending on the scenario, users can navigate from the line items to further details. Right-click on the table to display a contextual menu that enables you to do the following:

  • Copy the selected column or row
  • Paste a previously selected column or row
  • Delete the selected column or row
  • Add a column or row before the selected column or row
  • Add a column or row after the selected column or row
  • Clear the content of the select column or row

https://experience.sap.com/fiori-design-web/ui-components/responsive-table/
