Note The property pane also displays an Event tab to specify the navigation triggered when clicking the button.
Note A new page based on the List Report template contains the following by default:

  • One filter named Filter1

  • One column named Column 1, with Value="column1 value"

  • One column named Column2, with Value="column2value" and Additional Value="value2"
