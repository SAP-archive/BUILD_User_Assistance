Default Content in the Object Page Template

A new page based on the Object Page template contains the following by default:

  • In the header

    • ObjectType = ‘New Page’

    • Title = ‘Title’
    • Description = ‘Description’
    • One KPI, with Title = ‘KPI Title’, Value=’0.00’, Unit = ‘Unit’
  • One section named ‘Section1’, containing a Simple Form, with 2 Form elements (one being mandatory)
  • One section named ‘Section2’, containing a Multi-Form, with 2 groups named ‘Group1’ and ‘Group2’, containing 2 and 1 form elements respectively
  • One section named ‘Section3’, containing a Table with 2 columns ‘Column1’ and ‘Column2’
