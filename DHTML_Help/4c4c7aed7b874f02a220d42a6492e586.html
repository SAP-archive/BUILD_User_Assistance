UI Composer Overview

The UI Composer includes various tools to enable you to create beautiful and interactive mockups for your app.

These prototyping tools include:

Tool

Description

How To

Sidebar

The leftmost sidebar that includes icons to navigate to Home, Project, Research, Files, and Settings.

Use the show/hide sidebar icon across the toolbar at the top to show or hide the sidebar.

Toolbar

The main toolbar that appears across the top of the Prototype Builder.

Use this toolbar for displaying and hiding various items, going to the data modeler, selecting device, publishing, sharing, and more.

Left Toolbar

The leftmost toolbar that includes page information, UI elements, assets, data to use when creating prototypes.

Select available items to customize prototypes.

Pages

Displays pages and elements in a page, such as images, buttons, links.

Add pages to your prototype. Then create hot spots, buttons, lists, and more to make it more interactive.

UI Elements

Includes Open UI5 elements to use when creating prototypes. Elements are grouped in common areas, such as lists, patterns, user input, and so on.

You can search for a particular UI library or UI element

Assets

A library of files that you upload and use in the Prototype Builder.

Use the Files icon in the sidebar to upload images, documents, and media.

Data

Items belonging to the data modeler.

Profile Icon

Displays on the top right of the screen for managing your user profile.

Select the icon to customize your profile, change contact information, password, and to log out.

User Assistance Icon

Access user assistance to enable you to use Build more effectively.

Select the User Assistance icon to navigate to Build tutorials, videos, and user assistance.

Right Toolbar

The rightmost toolbar that includes asset and UI element properties, actions, and other settings.

Select an element or asset to customize the properties in the right toolbar.
