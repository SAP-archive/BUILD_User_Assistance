Add Objects from a Catalog

You add objects from a catalog by using the search feature.

  1. In the Data Modeler, click the Add button, then select From Search from the list.
  2. Type the name of the catalog or objects for which you are searching.
    The Data Modeler displays:

    • The name of the catalog and its tags

    • The name of the object found in the catalog

    • The total number of objects in the catalog

  3. From the results, select the required object or objects.
