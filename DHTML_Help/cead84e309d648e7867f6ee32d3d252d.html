Add Objects via Search

You add existing objects by using the search feature.

  1. In the Data Editor, click the Add button, then select From Search from the list.
  2. In the Search screen type the name of the catalog or objects for which you are searching.
    The Data Editor displays:

    • The name of the catalog and its tags

    • The name of the object found in the catalog

    • The total number of objects in the catalog

  3. Select the required object or objects.

    The objects are added to your canvas and all changes are saved.
