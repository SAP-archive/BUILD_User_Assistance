Add Properties to an Object

You add properties to an object from the panel on the right side of the panel.

  1. Click Add Property, then type the name of the property in the field. The naming rules for perperties are the same as those for objects.
  2. Select the type of the property from the list. The standard values are:

    • String - the default value

    • Numeric

    • DateTime

Note

You can set or remove a property key by using the key icon on the property.
