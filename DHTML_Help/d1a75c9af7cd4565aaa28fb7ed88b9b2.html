Administration

A BUILD administrator performs several tasks in the Administration Console.

Administrators can:
  • Set roles for and delete users
  • View logs
  • Add UI elements
User Roles

Users can be either standard users or guests. Standard users can create and manage projects whereas guests are invited only to participate in a study. When a user signs up to BUILD, he or she becomes a standard user. Guests are added as users when they are invited to participate in studies.

As an administrator, you can change the role of a user. Note however, that when you change a user's role, you delete all the properties that have been assigned to the user. You can search for a user either by name or by email address.

When you remove a user

You need to be careful when you remove a user because you also remove:

  • The projects owned by this user from the collection of projects
  • The user from project user list
  • The studies belonging to the deleted project from studies collection
  • Answers and annotations made by the user in studies in which he or she participated
  • The user and deleted projects from the role collection
  • The user from the collection of users
Audit Log
You can download the audit log from a pre-set or a custom date range. The pre-set date ranges are:
  • Last 24 hours
  • Last 48 hours
  • Last 10 Days
When you select a date range, the dates automatically appear in the Start and End boxes.

When you select Custom, you select your own start and end dates.
