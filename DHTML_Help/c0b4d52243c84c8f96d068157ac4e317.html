Create a Data Model

You begin creating a data model once you have created a project.

Context

If you enter the Data Modeler by clicking Go to Data Modeler, you land on the canvas where you create your objects. If you want to create a new data model:

Procedure

  1. Click inside the default object and give it a name.
  2. Add objects by clicking the Add button.
  3. On the right side of the canvas, click Relations, then assign a target object and cardinality.
