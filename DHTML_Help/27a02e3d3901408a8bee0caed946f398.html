The Object Page Template

The Object Page template contains a header and a collection of sections.

The Object Page template supports the following controls:
  • In the header:

    • KPI

    • Action button

  • In sections:

    • Simple Form

    • Multi-form
    • Table in a section

  • In Simple Forms: Form Element

  • In Multi-Forms: Groups
  • In Groups: Form Element
  • In Table: Column
Order of nodes in the Object Page template
The order of the nodes in the Outline is:
  1. Header
  2. KPI
  3. Action Buttons

    The order of Section nodes in the outline reflects the order of sections in the page. There can be only one Simple Form, Multi-Form, or Table in a section.

    The order of Group nodes under a Multi-Group, or of Form Element nodes under a Simple Form/Group reflects their respective order in the page.
