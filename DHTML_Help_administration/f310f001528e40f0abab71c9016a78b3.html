Set Domain Access

BUILD Admin can add domains and assign roles and permissions to each domain. Then users from these domains can be assigned roles and access BUILD.

Context

To add a new domain:

Procedure

  1. Select Access in the left sidebar. The Access List is displayed. On first access to the Admin Console, the default access level is set No Access.
  2. Select New Domain to add a new domain to BUILD and assign permissions. First you should add your own domain and give Full Access. Then you can add more domains.
  3. Enter the domain name in the format <domainname>domain.com, for example sap.com.
  4. Select the BUILD access level from the dropdown menu. You can choose from the following access levels:

    Access Level

    Permissions

    No Access

    Users cannot access BUILD at all.

    Study Invitation Only

    Users can access a BUILD study by invitation only as a Guest user.

    Study or Project Invitation Only

    Users can access a BUILD study OR project by invitation only. When the user is invited to the project, user becomes a Standard user so they can collaborate on the project.

    Guest With Study Invitation

    Users can sign in to BUILD as a Guest user and access the Study by invitation or by URL. When the user receives a study invitation from other users, they can access the study, as well.

    Guest With Study or Project Invitation

    User can sign in to BUILD as a Guest user and access the Study by invitation or by URL. When users are invited to a project, user become a Standard user so they can collaborate on the project.

    Full Access

    User has full access to all BUILD features
  5. Repeat these steps for all the domains you want to add.
