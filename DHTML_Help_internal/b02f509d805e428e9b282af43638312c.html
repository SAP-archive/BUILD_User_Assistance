Smart Templates

BUILD provides smart templates that help you design screens for your prototype.

You need to have either created or imported a data model in order to use Smart Templates. You create two types of pages when you use Smart Templates. The first page in you create is a list report. All subsequent pages are object pages.

Before you get started using Smart Templates to create list reports and object pages, take a few minutes to familiarize yourself with the BUILD Smart Template interface.

The templates contain the following sections:
  • Control Palette - displays a list of controls that are allowed by the Smart Templates.
  • Outline - displays the structure of the page. You can drag controls from the control palette to add content to the page. You can reorder items in the outline as well as remove items.
  • Property Pane - displays the properties of the object selected in the outline. You can then modify the values of these properties.
  • Preview Pane - read-only. Displays a preview of the template.
