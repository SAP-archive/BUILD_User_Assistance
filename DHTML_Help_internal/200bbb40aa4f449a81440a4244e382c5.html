First Steps for Administrators

As a BUILD Administrator, you need to perform these initial steps before others can use the system.

Context

You have received a Welcome email from the SAP BUILD team. Open the email, then click the Activate Me button in the email, which triggers the creation of your Cloud Identity. Once your Cloud Identity account is created, you are redirected to a 403 Forbidden page. Do not take this into account. Instead, proceed directly as follows to allow access at the domain level. For legal reasons SAP cannot do this on your behalf:

Procedure

  1. Click the Set things up in the Admin console button in the email that you received.

    Once you have logged in with the Cloud Identity, the BUILD Admin console page opens.

  2. Click Access in the toolbar on the left.
  3. Click New Domain, then set access for your domain users.
    Note

    Do not add domains such as gmail.com or yahoo.com as this would allow access to all gmail or yahoo users.

  4. Click Users in the toolbar, then Pick your user > Standard user and set yourself as a standard user.
  5. Send the system URL to your users. Once they have created their Cloud Identity user, they become BUILD standard users. Nothing else is needed from your side.

Next Steps

The BUILD Admin console allows the BUILD Admin to manage users, domains, and view audit logs. For detailed information about the BUILD Admin console see the Admin Console page.
