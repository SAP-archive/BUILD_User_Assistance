Create a Calculated Property

You can specify a formula for a property thereby creating a calculated property.

  1. Under the Properties tab, click New Property, then enter a name and select a type.

  2. Double-click the name of the new property then, in the next screen, click Formula > Create.

  3. In the Formula Editor, drag the property, value, operator and, if necessary, condition elements to the screen and select values.
  4. When you have finished, click OK.

    If valid, the syntax of the formula is displayed under the Formula tab. If the syntax is not valid, an error message is displayed.
