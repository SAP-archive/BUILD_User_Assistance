Create Relations Between Objects

After you have added objects to a data model you need to create relations between various objects.

Procedure

  1. Select the object from which you want to create the relation.
  2. In the panel on the right side of the canvas, click Relations> New Relation.
  3. Assign a target object and cardinality.

    The relation appears in the data model like this:

    Note

    You can also click the blue circle on the corner of the object and drag the arrow to another object to create a relation between objects.
