User Interface Editor

BUILD includes the User Interface (UI) Editor, which enables you to easily and quickly create beautiful prototypes, wireframes, and interactive click-through mockups for your application.

Create prototypes using images or HTML prototypes you can upload to BUILD. Prototypes can include interactive functions and actions and can be linked to a data model. Prototypes are easily shared for feedback, collaboration, and user research.
