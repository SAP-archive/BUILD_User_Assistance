Create a Calculated Property

BUILD allows you to specify a formula for a property. This property is a calculated property.

Procedure

  1. In the Data Editor, select Properties in the right side panel.
  2. Select New Property.
  3. In the DefaultName field, type a property name. Then select the property type from the dropdown list.
    Note

    The name of the new property cannot contain spaces.

  4. Select the Formula tab.

  5. Select Create.
  6. In the Formula Editor, drag formula elements, Property, Value, Operator, and if needed, Condition, to the area below, and select values for each element.
    Formula elements:

    • Property: Select an object of the model and one of its properties.

    • Value: Enter a value for this property.

    • Operator: Select an operator from the list:

      • Arithmetic operators: +, -, *, /

      • Comparison operators: <=, <, >=, >, =, !=

      • Logical operators: And, Or, IS NULL
    • Condition: Enter conditions in the formula syntax.

      • Three sections IF, THEN, and ELSE are automatically added to the canvas.

      • You can specify nested conditions.
      Note

      If the formula is valid, the syntax of the formula is displayed in the Formula tab. If it the formula is not valid, an error message appears.
