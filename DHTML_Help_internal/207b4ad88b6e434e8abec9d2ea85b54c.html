Add Prototype Pages

Add pages in the UI Editor to start creating your application.You can add images to the UI Editor canvas and start creating an interactive prototype. You can also add template pages with predefined properties and controls.

Prerequisites

Before you can start creating pages, you must first start a new project for your application See tCreate Projects.You can to upload images to your project or simply drag and drop images from your computer.

If another team member is editing the page you are working on, the page displays as Locked, Editing. When the other team member finishes editing the page, you can edit the page.

Context

To add prototype pages:

Procedure

  1. Open your BUILD project.
  2. Open the Page Map and select Add to add a Blank Page. To add more pages, select New Page in the page thumbnail view.
  3. Open the page you want to start designing. The UI Editor canvas displays. For details on the UI Editor tools, see User Interface Editor Overview
  4. Use the page map view at the top left to rename and delete pages.
    • Rename Page: To rename a page, select the page name (for example, Page 1) in the page map, select the edit button to rename the page.
    • Delete Page: To delete a page, select the page name the page map, select the delete button . Confirm the page delete. All content on that page is deleted.
    • Add Page: Select +New Page to add more pages to your prototype.
  5. Add an image to your canvas.
    • Drag and drop an image from your computer to the canvas.
    • Drag and drop an uploaded image from the Assets tab in the UI Editor to the canvas.
    • Upload an image using the Files icon in the left navigation bar. See Upload Files for details.
  6. Select the image, then select the Zoom icon % on the top toolbar to change the image size.
    • Size 1:1: The image resizes to 100%, even if it does not fit in the current canvas size. You can scroll to view the rest of your image.
    • Fit Width: The image resizes to fit the canvas.
    • Zoom: drag the zoom bar to the image size you need.
  7. Now you can begin adding UI Controls to make your prototype interactive. See Add Controls for instructions.
