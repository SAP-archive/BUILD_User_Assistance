Add Blank Pages

Add pages in the UI Editor to start creating your application. You can add responsive pages for various platforms, blank freestyle pages, or pre-defined template pages to your prototype. Add blank pages to create freestyle prototype pages. Blank indicates an open canvas where you can add whatever you like on the canvas.

Prerequisites

Before you can start creating pages, you must first start a new project for your application. See Create Projects.You can upload images to your project or simply drag and drop images from your computer.

If another team member is editing the page you are working on, the page displays as Locked, Editing. When the other team member finishes editing the page, you can edit the page.

Context

To add prototype pages:

Procedure

  1. Open your BUILD project.
  2. Open the Page Map, select Add in the Blank Page box, then open the page. For details on the UI Editor tools, see User Interface Editor Overview. You can also add one of these pages:
    • Responsive Page: Build cross-platform apps in a responsive page layout for use on various platforms, phone, table, and desktop. You can mix blank pages with responsive pages in your prototype. See Add Responsive Pages for more information.
    • List Report Template: Use smart templates to quickly create pre-defined template pages. See List Report Page for more information on using the List Report template.
  3. Use the page map view at the top left to rename, delete, and and add pages:
    • Rename Page: To rename a page, select the page name (for example, Page 1) in the page map to enter a new page name.
    • Delete Page: To delete a page, select the page name on the page map, and select Delete on your device. All content on that page is deleted. To undo a delete, select the undo button on the top toolbar.
    • Add Page: Select +New Page to add more pages to your prototype. Select either Blank Page or Responsive Page.
  4. Use the % magnification icon on the toolbar to resize the canvas. Adjust thesize by selecting Zoom, Size 1:1, and Fit Width on the top toolbar to resize the canvas.
  5. Add an image to your canvas.
    • Drag and drop an image from your computer to the canvas.
    • Drag and drop an uploaded image from the Assets tab in the UI Editor to the canvas.
    • Upload an image using the Files icon in the left navigation bar. See Upload Files for details.
  6. To resize the image, select it, and use the Scaling properties on the right panel.
  7. Select the Preview button to preview your prototype pages.
  8. Now you can begin adding UI Controls to make your prototype interactive. See Add Controls for instructions.
