Add Properties to an Object

You add properties to an object from the panel on the right side of the panel.

  1. Click Add Property, then type the name of the property in the field. The naming rules for properties are:

    • Maximum length of 40 characters

    • The first character must be a letter or an underscore

    • Subsequent characters must be alphanumeric or an underscore

    • The name must be different from names of other properties of the object
  2. Select the type of the property from the list. The standard values are:

    • String - the default value

    • Numeric

    • DateTime

Note

You can set or remove a property key by using the key icon on the property.

Modifying Data Under the Samples Tab

BUILD allows you to modify data by editing it directly in the application or by importing or exporting files to and from Excel. These actions are performed under the Samples tab.

Importing and Exporting Objects and Models to and from Excel

BUILD allows you to import and export objects and models to and from Excel from the Samples tab. When exporting, the Excel file takes the name of the project. It takes the name of the project and is always a table with either mapping tables or foreign keys. You can choose to export a selected object or the entire model. You can also select Relations with Tables which means that the relations are identified by mapping tables. If you do not select Relations with Tables, the relations are identified by foreign keys.

When importing, you can either drag a file from your local drive or browse to choose a file. You can import one of three Excel formats:

  • Data sheet with foreign keys

  • Table with foreign keys
  • Table with mapping table
Note

The structure of the file that you import must correspond exactly to the structure of the data model in BUILD.

Editing Data in the Data Model
You can edit data in the data model in BUILD using the editor under the Samples tab.
  1. Select the object for which you want to edit data, then click Open Editor.
  2. Click inside the cells for which you want to modify the data, then click OK.
