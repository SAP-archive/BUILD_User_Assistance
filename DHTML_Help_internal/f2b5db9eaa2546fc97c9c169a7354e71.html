Add a List Report

Once you have created or imported a data model you can start using the smart templates.

Procedure

  1. From your data model, click the paintbrush icon .
  2. Under ListReport Smart Template click Add.

  3. By default you land on the New Page screen. Click the default page proposed, Page 1.
  4. The ListReport section under Properties is pre-populated with properties from your data model. Select the object that you want to use as your Main Object. Each list report and object page has its main object. When you select the main object you bind the list report to the object.
  5. Drag the controls from the Control palette to the outline to create fields in the List Report, for example, add a second filter or an additional column to the table.
  6. Select elements in the Outline and enter values for the properties.
    Note

    You must be in the Outline pane to be able to enter values for the properties

    Example of a filter, Product Name, bound to the Name property of the Product object.

  7. Add table objects (action buttons and columns) where necessary.

    Example Action button

    Note

    At any time, you can click the Preview button to preview your list report.

  8. When you are ready to populate your list report, click Go.
