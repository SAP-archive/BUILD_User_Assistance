Add Controls

Use the UI library controls in the UI Editor to make prototypes interactive. Drag and drop controls to the canvas, and apply properties and actions to customize their function.

Prerequisites

Before you can add UI controls to a prototype, be sure you have an image on a page in the UI Editor.

Context

To add UI controls to your prototypes:

Procedure

  1. Open the UI Editor and one of your prototype pages.
  2. Select a control from the Controls tab on the left. Drag the control onto the canvas and place it on your image. For example, select Button or Hotspot.
  3. With the control still selected on your image, you can change the Properties and Actions displayed on the right of the canvas.
    • Properties: Properties control how the UI element looks. Select the UI element and change the properties to meet your requirements.
    • Actions: Actions control what the UI element does when the user selects it. Select the UI element and change how you want the UI element to interact.
    Note

    Check out the Open UI5 Developer Guide and the UI5 Controls Reference for more information.

  4. Select the Preview button on the toolbar to preview and test interactions in your prototype. Previewing your prototype displays the prototype pages and interactions as users will see them. Select your buttons, and other controls to see how they respond.
  5. Select the Editor button on the toolbar to return to the UI Editor.

Next Steps

Publish Prototype: If you want to publish and share your prototypes with the team for collaboration, select the Publish icon on the toolbar. Then select the Publish button. Select the URL and share with others.

Create Study: If you want to create a user study to get feedback on your prototype, select the Create Study icon on the toolbar. Add a study name and description and Create and Go to Research. See Create User Research Studies for details on creating user research studies.
