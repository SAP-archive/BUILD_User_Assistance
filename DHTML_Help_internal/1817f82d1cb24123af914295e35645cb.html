Add Objects from an Excel File

You can add objects from an Excel file.

  1. Click Add, then select Import xls from the menu.

  2. Browse to locate the file you want to import, then import.
  3. The objects are displayed with their properties and sample data in the canvas of the Data Modeler.

BUILD provides sample Excel files to help you create data model.
