Page Map

View the Page Map from th UI Editor to see a map view for your project. Page thumbnails and navigation are also represented in the Page Map.

Context

To use the Page Map:

Procedure

  1. Select the Page Map icon from the UI Editor toolbar.
  2. In the Page Map, you can:
    • The Home page is indicated by the icon.
    • View thumbnails of all pages in your project.
    • View all the connections between pages in your project.
    • Add and delete pages
  3. Select a page thumbnail to open it in the UI Editor.
