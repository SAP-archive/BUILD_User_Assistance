Editing Data in the Data Modeler

You can edit data in the Data Editor using the Samples tab.

Procedure

  1. From the Data Editor, select the object you want to edit, then select Open Editor.
  2. Make changes in the table cells or header.
  3. If necessary, select the + plus sign in the upper left of the table to add new rows.
  4. When you are finished with changes, select OK.
