Editing Data in the Data Modeler

You can edit data in the data model in BUILD using the editor under the Samples tab.

Procedure

  1. Select the object for which you want to edit data, then click Open Editor.
  2. Make any modifications in the cells or header of the table.
  3. If necessary, click the plus sign in the upper left corner of the table to add empty rows.
  4. When you have completed your modifications, click OK.
