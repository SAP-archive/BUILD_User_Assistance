Object Page

The Object Page displays all relevant information about an object. The Object Page template contains a header and a collection of sections.

As with the list report template, BUILD lets you add controls and sections by dragging them to the Outline pane; you add properties in the Properties pane.

Note

As you add controls and properties, the changes are displayed in the Preview pane. You cannot make changes in the Preview pane, however.

Preview pane of the object page

The Object Page template contains the following controls:
  • In the header:
    • KPI
    • Action button
  • In sections:
    • Simple Form
    • Multi-form
    • Table in a section
  • In Simple Forms: Form Element
  • In Multi-Forms: Groups
  • In Groups: Form Element
  • In Table: Column
