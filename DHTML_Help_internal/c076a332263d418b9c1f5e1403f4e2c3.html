Publish Prototype

You an publish a prototype at any time and share it with others. This creates a "snapshot" in time for the prototype design.

Context

From the Prototype page:

Procedure

  1. Select the Publish icon in the toolbar. A shareable link displays with a specific URL address.
  2. Select Publish Project. Copy the link and share with others via email.
