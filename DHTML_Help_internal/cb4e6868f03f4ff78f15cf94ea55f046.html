Control Properties in the Object Page Template

This table summarizes how properties are displayed in the property pane:

Location

Property Name

Description

Header

Header Bar Text

The text display in the header bar at the top of the page. Bindable.

Header

Title

Title of the object page, usually the name/ID of the current object. Bindable.

Header

Description

Can be bound to the data model

The description of the Object Page Bindable.

Header

 Image Source

The URL pointing to the image that will be displayed in the header. Bindable.

KPI

Title

The title of the KPI, displayed at the top of the KPI.

Displayed in the outline

KPI

Value

The value of the KPI, displayed on the left. Bindable.

KPI

Currency

The currency or the unit for the KPI, displayed on the right; Bindable.

Action Button

Label

Label of the action button.

Displayed in the outline.

Section

Title

The title of the section

Displayed in the outline

Form Group

Title

The title of the form group. Displayed in the outline.

Form Element

Label

The label of the form element

Displayed in the outline

Form Element

Value

The value of the form element. Bindable.

Form Element

Mandatory

Specifies if the element is mandatory in the form. If "Yes" an asterisk is displayed left of the label

Form Group

Title

The title of the form group.

Displayed in the outline.

Column

Title of the column.

Displayed in the outline. Not bindable.
Column

Value

Value displayed in the column. Bindable.

Column

Currency

 If this property is specified, its value is concatenated after Value and is displayed as a currency. Bindable.

Column

URL

 Allow navigation to another ObjectPage via proposing the navigation links from the object bound to the table. Bindable.

Column

Additional value

 A second value is to be displayed in the column. If specified, this value is displayed in bold typeface above the first value.

Column

Importance

 Specifies the responsive behavior of the column:

  • High: the column is always displayed

  • Medium: the column pops in when the size decreases

  • Low: the column disappears when the size decreases
Not bindable

Column

Show sub-totals

Default value is No.

When this value is set to Yes, the content of this column is aggregated when grouping fields in the smart table properties.
Note No properties are displayed in the Property pane when you select a Simple Form, Multi-Form, or Table node in the outline,

Column: See Column in the Control Properties of the List Report.
