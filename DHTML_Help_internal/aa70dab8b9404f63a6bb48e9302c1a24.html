Control Properties in the List Report Template

This table summarizes how properties are displayed in the property pane:

Control

Property Name

Description

Filter Bar

Allow Free Search

Specifies if the Free Text Filter box is displayed. Not bindable.

Filter

Description

Label of the search box for the filter.Not bindable.

Filter

Filter on

Name of the object property to filter on. Bindable.

Table

Title

Object name displayed in the table header. Not bindable.

Action button

Label

Label of the action button.

Displayed in the outline. Not bindable.

Column

Title

Title of the column. Displayed in the outline. Not bindable.

Column

 Value

Value displayed in the column. Bindable.

Column

Currency

If this property is specified, its value is concatenated after Value and is displayed as a currency. Bindable.

Column

URL

Allow navigation to another ObjectPage via proposing the navigation links from the object bound to the table. Bindable.

Column

Additional value

A second value is to be displayed in the column. If specified, this value is displayed in bold typeface above the first value.

Column

Importance
Specifies the responsive behavior of the column:

  • High: the column is always displayed

  • Medium: the column pops in when the size decreases

  • Low: the column disappears when the size decreases
Not bindable.

Column

Show sub-totals

Default value is No.

When this value is set to Yes, the content of this column is aggregated when grouping fields in the smart table properties.
Note The property pane also displays an Event tab to specify the navigation triggered when clicking the button.
