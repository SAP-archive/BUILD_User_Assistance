Add a Smart Template

If your UI catalog contains Smart Templates you can add them.

  1. From the Prototype pages click View all: Map, then click the Add Page button.
  2. Select the type of template you want to add, either:
    • Object Page
    • List Report

The Smart Template Library displays a list of controls that are allowed by the Smart Template. This list is specific for each template. These controls are organized in groups to facilitate their usage. The Outline displays the structure of the page.

The types of controls you can drag to the outline depend on the template used for the page. You can reorder elements in the outline, for example, the order of sections or the order of action buttons.

The Property pane displays the properties of the object selected in the outline. You can modify the values of these properties, and bind them with the data model. The Preview pane shows how the page will look when finished. The preview area is read-only, which means that you cannot drag controls to it.

Note

You cannot mix free-form templates with smart templates. Smart templates can be followed by object list or object page templates.
