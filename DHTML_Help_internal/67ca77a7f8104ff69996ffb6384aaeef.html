Add an Object Page

After you configure your List Report, you are ready to configure one or more object pages.

Procedure

  1. From the Outline select +New Page >Object Page Smart Template.

    The object page, Page 2, opens. By default, the object page contains the following:

    • Header

    • Form section

    • Table section

  2. Select Page 2, then under Properties select the Main Object. The main object is usually the same as the main object of the list report.
  3. then
    • In the Outline select Header.
    • Under Properties configure the the title and the description of the object page.

      • Header bar text

      • Title
      • Description of the object page.

    You can type a custom value or select one from the list.

  4. If you want to add a KPI:
    • Select KPI in the Outline.
    • Under Properties, type a title for the KPI.
    • Select a value from the list.

    If you don't want to add a KPI, you can delete it in the Outline.

  5. Expand the first section in the outline, select Form group Group 1, then, under Properties, configure the form group with its form element labels. You can have more than one value for each label.
  6. Expand the second section and select Table.
  7. Under Properties select the Item that you want displayed in the rows of the table
  8. Under Outline, select each column, then under Properties, type a title for each column and select values for the column.
  9. Select the Preview button to view your object page with its values.

    This object page contains the following elements:

    • Header bar: Best Sellers

    • Header title: Product name, Midnight Fighter
    • Description: ID number, P32191
    • One section: Information
    • One form group in the section: Reviews
    • Two form elements in the form group: Number Reviews and Notes
    • A second section: Individual Sales
    • A table in the second section that displays the product ID, the part name, and the category
  10. If you want to continue adding or editing data in the object page, select Editor.
    Note

    As with all forms, there is no need to save your work. BUILD automatically saves it for you.

    You can preview your object page at any time by selecting the Preview button.
