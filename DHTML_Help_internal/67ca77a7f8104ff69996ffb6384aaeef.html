Add an Object Page

When you have finished configuring your List Report, you are ready to configure one or more object pages.

Procedure

  1. From the Prototype Pages click Page Map, then New Page.

  2. Use the controls in the Controls palette to configure the header and sections of the new object page.
  3. Select the sections in the outline and enter values for these sections under the Properties tab.
    Note

    As with all forms, there is no need to save your work. BUILD automatically saves it for you.
