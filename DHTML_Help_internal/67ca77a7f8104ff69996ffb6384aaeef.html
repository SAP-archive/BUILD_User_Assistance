Add an Object Page

When you have finished configuring your List Report, you are ready to configure one or more object pages.

Procedure

  1. From the List Report page click the Page Map icon New Page.

    By default an object page, Page 2, appears in the canvas.

  2. Click Page 2.

    The Object Page template opens.

  3. Under Properties select the Main Object. The main object is usually the same as the main object of the list report.
  4. Use the controls in the Controls palette to configure the header and sections of the new object page.
  5. Select the sections in the outline and enter values for these sections under the Properties tab.
    Note

    As with all forms, there is no need to save your work. BUILD automatically saves it for you. You can preview your object page at any time by clicking the Preview button.
