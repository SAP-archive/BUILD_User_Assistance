User Interface Editor Overview

The User Interface Editor includes various tools to enable you to create beautiful and interactive mockups for your app.

The following image displays each part of the UI Editor with descriptions in the table below.

Tool

Description

Navigation Sidebar

The leftmost navigation toolbar that includes icons to navigate to Home, Project, Research, Files, and Settings.

Page Elements

Displays pages and elements in a page, such as images, buttons, links, and New Page button. Select each page to expand its contents. Here you can view, rename, add, delete pages. You can also view page elements.

UI Controls

The left toolbar in the UI Editor that t includes page map, controls, assets, data.

  • Controls: Open UI5 elements to use when creating prototypes. Elements include button, list, calendar, table, toolbar, and more.. You can also select a UI library and search for specific UI elements.
  • Assets: Assets or Files (images, documents, prototypes, that you upload to BUILD to use with your prototype.
  • Data: Access data modeling to work with the data model, bind data to pages, and load sample data.

UI Editor Toolbar

The main toolbar that appears across the top of the UI Editor. Use it to change the image on the canvas (zoom, fit, device type), share images or prototypes, or create a user study.

Use the Preview button to see your interactive prorotype. Then go back to the UI Editor with the Editor button.

UI Editor Canvas

The center area of the UI Editor where you can design your beautiful prototypes. Drag and drop images, import images, and customize prototypes with interactive controls.

Help Icon

Select the user assistance (Help) icon to access step-by-step instructions, videos, and more content to help you learn to use BUILD.

Control Properties

Customize the look and feel of controls in your prototype.

Control Actions

Customize the interactions of controls in your prototype
