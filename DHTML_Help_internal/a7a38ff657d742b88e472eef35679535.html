Rules for the Object Name

You'll need to observe several rules when naming objects.

The rules for the object name are:
  • Maximum length = 40 characters
  • The first character must be an alphabetical character or an underscore "_"
  • Subsequent characters must be alphabetical characters or underscores "_"
  • The object name may not be the same as another object name in the data model.
  • Some names such as C or R, which are Excel reserved names, are not allowed
