Add Objects by Search

You can add objects from business catalogs by using the Search feature.

Context

From the data model, select the search link or select Add >From Search from the list.

Procedure

  1. In the Search for Data Objects screen, type the name of the object or catalog that you are searching for.

  2. Select one of the objects found in the business catalog.
  3. In the next screen, select what you want to add. You can add just the object or the object and its relations. If you select the object and its relations, you import the object and its children.
