Add Objects to a Data Model Manually

You begin creating a data model once you have created a project. This section describes how to manually create a data model by adding objects.

Context

When you enter the Data Editor, you land on the canvas where you add objects.

If you want to create a new data model from scratch:

Procedure

  1. In the Data Editor do one of the the following:
    • Click the create new objects link
    • Click the plus sign icon and select Manual from the menu.

  2. Type the name of the new object.
  3. If necessary, add more objects by clicking the Add button and selecting Manual from the menu.

    After you've added objects, you need to add properties to the objects and create relations between objects.

    Note

    You may notice that there is no Save button. BUILD saves all your changes automatically.
