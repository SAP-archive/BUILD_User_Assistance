Add Responsive Pages

Add pages in the UI Editor to start creating your application. Add responsive pages to build cross-platform apps in a responsive page layout for use on various formats, phone, table, and desktop. Responsive pages automatically place controls according to the format (phone, tablet, desktop).

Prerequisites

Before you can start creating pages, you must first start a new project for your application. See Create Projects.You can to upload images to your project or simply drag and drop images from your computer.

If another team member is editing the page you are working on, the page displays as Locked, Editing. When the other team member finishes editing the page, you can edit the page.

Context

To add responsive pages:

Procedure

  1. Open your BUILD project.
  2. Open the Page Map, select Add in the Responsive Page box, then open the page. For details on the UI Editor tools, see User Interface Editor Overview. You can also add one of these pages:
    • Blank Page: Starting your project with a blank page is ideal for freestyle prototyping. You can mix responsive pages with blank pages in your prototype. See Add Blank Pages for more information.
  3. Use the page map view at the top left to rename and delete pages.
    • Rename Page: To rename a page, select the page name (for example, Page 1) in the page map to enter a new page name.
    • Delete Page: To delete a page, select the page name the page map, and select Delete on your device. All content on that page is deleted. To undo a delete, select the undo button on the top toolbar.
    • Add Page: Select +New Page to add more pages to your prototype. Select either Blank Page or Responsive Page.
  4. Use the magnification icon on the toolbar to resize the canvas. Use Zoom, %, Size 1:1, and Fit Width on the top toolbar to change the canvas to the size you prefer.
  5. Add an image to your canvas.
    • Drag and drop an image from your computer to the canvas.
    • Drag and drop an uploaded image from the Assets tab in the UI Editor to the canvas.
    • Upload an image using the Files icon in the left navigation bar. See Upload Files for details.
  6. To resize the image, select it, and use the Scaling properties on the right panel.
  7. Customize your page on the right panel. Add Page headers, footers, and enable scrolling under Properties. Select Add under Children in the right panel to add the controls you need to your page.
  8. Select phone, tablet, or desktop icons on the toolbar at the top to preview your prototype in that format.
