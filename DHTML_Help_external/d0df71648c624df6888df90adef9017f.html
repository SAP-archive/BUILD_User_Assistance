Edit a Relation

You can edit a relation from the Relations tab in the right side panel of the Data Editor.

Procedure

  1. Under the Relations tab, select the relation that you want to edit.

  2. In the Info dialog box, make the required changes. You can make changes to the following:
    • Name
    • Target object
    • Cardinality
    • Read/write status

      You can also view the name of the foreign key used in the model to manage the relationships between instances of the two objects in the relation.
