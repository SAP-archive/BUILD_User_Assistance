UI Composer

The Prototype Composer allows you to easily and quickly create beautiful wireframes and click-through mockups for your app. .

Prototypes can include interactive functions, boxes, images, videos, text, and more. Prototypes can also be linked to a data model and shared for evaluation and user testing. Before creating prototypes, you must first create a Project. In addition, you can upload HTML prototypes to use in user research.
