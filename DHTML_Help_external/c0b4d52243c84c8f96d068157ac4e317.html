Create a Data Model

You begin creating a data model once you have created a project.

Context

When you enter the Data Editor, you land on the canvas where you create your objects.

If you want to create a new data model from scratch:

Procedure

  1. Click inside the default object and give it a name.
  2. Add objects by clicking the Add button.
  3. Select the object, then in the panel on the right side of the canvas, click Relations, then assign a target object and cardinality. You can also click the blue circle on the corner of the object and drag the arrow to another object.

    Note

    You may notice that there is no Save button. BUILD saves all your changes automatically.
