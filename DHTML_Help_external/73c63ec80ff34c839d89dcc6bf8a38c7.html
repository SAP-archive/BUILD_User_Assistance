Prototype

The Prototype tab of the participants responses to a task displays the fully-interactive prototype upon which the task was based, and displays participants' responses to each page of the prototype and to the overall task.

Fig. 5.4.3.3.1.

Prototype

You can do the following on the Prototype tab:

  • Interact with the prototype. You can do this by navigating through and using the prototypes itself, or by clicking the PREVIOUS or NEXT button in the right-hand panel.

  • Click the Show Annotations button to do the following:

    • View all annotations icons dropped on the page. The page fades significantly so that you can view the annotation icons more easily.

    • View all the comments entered about this task in the COMMENTS tab on the right-hand panel. If you click a comment, a line connecting the comment to the anntation icon in which it was entered is displayed. In addition, you can filter the comments entered by sentiment.
      Note

      You click the Hide Annotations button to hide the annotations.

    • Click the Show Heatmap button to show a heatmap that represents the frequency and location of user-clicks on the page of the prototype currently displayed.
