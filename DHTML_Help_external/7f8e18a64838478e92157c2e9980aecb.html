Inviting Team Members

You can invite your colleagues to join a project you created or projects you are a team member of.

Procedure

  1. On the Home page, select the tile for the project.

    The Project page is displayed.

  2. Under Project Team, select the Add People button.

    The ADD PEOPLE dialog is displayed.

  3. Enter the email address of a colleague, and select ADD.

    The email address is added to the LIST OF INVITEES section.

  4. You can invite more colleagues by entering an email address and selecting ADD for each.
  5. Select the Send Invite button.

Results

Colleagues you invite to join the project are sent an email inviting them to join the project. For more information on accepting invitations to join a project, see Accept an Invitation.
