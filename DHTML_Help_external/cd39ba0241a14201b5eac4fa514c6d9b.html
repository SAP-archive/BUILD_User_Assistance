Create Projects

When you use BUILD you are always working on a project. You can create multiple projects in BUILD. In each BUILD project you can invite other colleagues to become contributing team members, and create user research studies specific to the project.

Context

You must have created a BUILD account and logged on to BUILD before you can create a project.

Procedure

  1. Select the new project link or the New Project link on the Home page.

    A new project tile is displayed on your Home page.

    Note

    The new project link is not available if you have previously created a project or become a team member of another project.

  2. Enter a name for the project, and press Enter on your keyboard.

    The project is created and the project tile is displayed in the Project section on the page. The Crown icon displayed on the project tile on the Home page indicates that you are the owner of this project.

  3. Select the project tile to display the project.

    The project is opened and the Project page is automatically displayed.

    The Crown icon displayed on your user icon in the Project Team section on the Project indicates that you are the owner of this project.

    On the Project page, you can create user research studies and invite collegaues to join the project.

    In addition, you can navigate through the application by:

    • Selecting the Home button to return to the home page;

    • Selecting the Research button to view the research page;

    • Selecting the Files button to upload files to a project;

    • Selecting the Settings button to change your project's settings.
