Example: Exporting a Modified Model

In the previous example workflows, you have modified data in an object. In this workflow, you export the model to Excel.

Procedure

  1. With your modified model open, select Samples > Export.
  2. Select Data Model to export the entire model.
  3. Leave the default value, Relations with Tables, selected in order to keep the structure of the data with its mapping tables.
  4. Select Export. The entire data model is downloaded to the default location on your computer.

    When you open the spreadsheet, you can see the object that you added, Supplier, with the properties and data that you added to the data model: ID, SupplierName, and EmailAddress.
