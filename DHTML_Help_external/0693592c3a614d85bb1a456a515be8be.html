View Audit Log

BUILD Admins can use the Admin Console to view and download user audit log files.

Context

To view and download audit log files:

Procedure

  1. Select Audit on the left sidebar.
  2. Select a date range: Last 24 Hours, Last 48 Hours, Last 10 Days, or select a Custom date range from the calendar.
  3. Select Download. A log file displays in .csv (comma separated value file). Save it to your computer and view it for details on user access, permissions, and more.
