Smart Templates

BUILD provides smart templates that help you design screens for your prototype.

When you create a new page in a project, of course you can start from a blank page, however you may prefer to use one of the templates provided:

  • Object Page

  • List Report

You use Smart Templates after you have designed a data model.
The templates have the same layout:
  • Control Palette - displays a list of controls that are allowed by the Smart Templates.
  • Outline - displays the structure of the page. You can drag controls from the pane to add content to the page. You can reorder items in the outline.
  • Property Pane - displays the properties of the object selected in the outline. You can then modify the values of these properties.
  • Preview Pane
