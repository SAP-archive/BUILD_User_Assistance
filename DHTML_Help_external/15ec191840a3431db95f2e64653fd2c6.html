Change Project Owners

The current owner of a project can make any member of a project an owner. Only project owners can archive projects.

Procedure

  1. On the Home page click the project tile of the project you want to change ownership of.

    The Project page is displayed. An avatar for each project team member with their name below is displayed in the Project Team section. Notice the Crown icon on your avatar.

  2. Drag-and-drop the Crown icon from your icon onto another team members icon to make them the own.

    Only project owners can archive projects, which can be done in the Settings page.
