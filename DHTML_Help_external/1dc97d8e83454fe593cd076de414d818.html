Extend Prototypes with SAP Web IDE

SAP Web IDE is a web-based tool that enables you to create and extend end user applications for browser and mobile devices. It simplifies the end-to-end application life cycle: prototyping, development, packaging, deployment, and customer extensions for SAPUI5 applications.

Prerequisites

Before working in Web IDE, you must first:

  • Create a prototype in the BUILD application.

    Depending on what type of pages you add to your prototype, see the following sections for more information::
    • Add Blank Pages
  • Publish the prototype. See Publish Prototype
  • Import the Smart Template Excel file into the Data Editor. You can find this file here: Github.
  • Set Run Configurations Run Prototype in Web IDE in Web IDE .

Context

Note

For more information on using Web IDE, see SAP Web IDE, see the SAP HANA Cloud Documentation.

Procedure

  1. Open the HCP IDE using the link provided to you by SAP.
  2. Enable the BUILD plugin:
    1. Select  Tools  Preferences .
    2. Under Plugins, Optional Plugins, select BUILD Plugins under the Plugin Repository. Then select Save.
  3. From the Home page or the  File  New  menu, select New Project from Template.
  4. On the Template Selection page, select Build Project, then select Next.
  5. Enter a Project Name, then select Next. The project name must start with a letter or underscore and can contain letters, numbers, periods, and underscores. The must not end with a period.
  6. On the Select BUILD prototype page, select Build System source, select your available prototypes, then select Next.
