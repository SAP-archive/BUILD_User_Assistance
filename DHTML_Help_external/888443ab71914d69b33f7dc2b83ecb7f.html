Create Relations Between Objects

After you add objects to the data model, you must create relations between objects.

Procedure

  1. In the Data Editor, select the object that you want to create a relation from.
  2. In the right side panel, select Relations, then + New Relation.
  3. Type a name for the relation.
  4. Select a target object from the dropdown list, and then select the cardinality, 1 or n.

    Note

    You can also select the blue circle on the corner of the object and drag the arrow to another object to create a relation between objects.
