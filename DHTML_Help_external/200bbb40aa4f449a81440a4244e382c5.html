Getting Started for Administrators

As a BUILD Administrator, you need to follow these initial steps before users can use the system.

Context

To enable users to use the BUILD system:

Procedure

  1. Select the Set things up in the Admin console in the Welcome email.

    After you log in with the Cloud Identity, the BUILD Admin console page opens.

  2. Select Access in the toolbar on the left.
  3. Verify that your domain access is listed and configured. Admin access level for your domain is automatically configured. Your BUILD users need Full Access instead of Guest With Study or Project Invitation. Change access levels and add new domains or as needed.
    Note

    Do not add domains such as gmail.com or yahoo.com as this would allow access to all Gmail or Yahoo users.

  4. Select Users in the toolbar, then lelect your user name and make sure you are set as a Standard User (if you need this access level).
  5. Send the system URL to your users. After users create their Cloud Identity user, they are automatically BUILD standard users. No additional admin changes are necessary.

Next Steps

The BUILD Admin console allows the BUILD Admin to manage users, domains, and view audit logs. For detailed information about the BUILD Admin console see the Admin Console page.
