Add a Project Picture

The project owner you can display a picture on the project tile. Adding a unique picture to each of your project tiles makes it easier to identify the project at a glance.

Context

You select the project picture from the images you have uploaded to the project. You can upload images on the UI Composer, the Research page, and the Files page.

Procedure

  1. Select the project tile on the Home page to open the project.
  2. Select Settings on the Project page.
  3. Select Project Picture.
  4. Select SELECT IMAGE.
  5. Select the image you want as a the project picture.
  6. Select SELECT.

Results

The image is displayed on the project tile. All invited members also see the picture.
