Importing and Exporting Sample Data

BUILD allows you to import data for existing objects and export objects and models to a spreadsheet from the Samples tab.

Importing

When importing, you can either drag a file from your local drive or browse to choose a file. You can import one of three formats:
  • Data sheet with foreign keys
  • Table with foreign keys
  • Table with mapping table
Note

The structure of the file that you import must correspond exactly to the structure of the data model in BUILD, that is, the objects with their properties and relations.

Exporting

When exporting, the Excel file takes the name of the project. The file is always a table with either mapping tables or foreign keys.

You can choose to export a selected object or the entire model. You can also select Relations with Tables which means that the relations are identified by mapping tables. If you do not select Relations with Tables, the relations are identified by foreign keys.
