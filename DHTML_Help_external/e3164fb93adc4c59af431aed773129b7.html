Example: Manually Adding Data

You can add data to an object in your data model manually

Context

In this example you add data to the Supplier object that you created in the previous workflow.

Procedure

  1. Select the Supplier object, then Select the Samples tab.

  2. Select Open Editor. In the editor, the name of the object, Supplier, is in the tab at the top of the page.

  3. Select the plus sign below the Supplier tab. This adds a line to the table.

    Notice that the properties of the Supplier object appear in the column headers of the table: ID, SupplierName, and EmailAddress.

  4. Type information in the table:
    • S1 for ID
    • Supplier One for SupplierName
    • Bob@example.com for EmailAddress

  5. Select OK. You have added data to your object and you are ready to export it.
