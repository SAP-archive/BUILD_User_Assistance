Manage Users

BUILD Admins can use the Admin Console to view and delete users, change roles.

Users and Roles
To view users:
  1. Select Users in the left sidebar. BUILD users are displayed.
    Change the user view using icons on the top right:
    • Select to view user summaries.
    • Select to view user details.
  2. To search for a specific user, enter the user name or email address in the search bar at the top.
  3. Select a user to view their role.
  4. Check the appropriate user role and select OK.
    • Standard: BUILD user can create and manage projects
    • Guest: BUILD user can only participate in user studies
Note If the Admin also wants to access BUILD, he should assign himself the Standard user role.
Note When you change a user's role from Standard to Guest user, the following are affected as well:
  • Deletes projects user owns
  • Deletes studies and all study feedback associated with deleted projects
  • Deletes user from a project
Delete Users
Note Use caution when deleting users. When you delete a user, the following are affected as well:
  • Deletes projects user owns
  • Deletes studies and all study feedback associated with deleted projects
  • Deletes user from a project
  1. Select the icon next to the user (summary or detail view) to delete a BUILD user.
  2. Confirm you want to delete this user and all project-related access. Select Delete.
