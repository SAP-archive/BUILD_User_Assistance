Archive Projects

As owner of a project you can archive and restore a project on the Project page. It is not possible to modify archived projects. You can restore archived projects anytime.

Prerequisites

You must have created at least one project.

Procedure

  1. On the Project page, select Archive Project.
    The Archive Project dialog is displayed.
  2. On the Archive Project dialog, select Archive.

Results

The project is archived, and is no longer displayed on your Home page. You can view a list of your archived projects by selecting archived on the Home page. You and other team members cannot modify archived projects. In addition, people can no longer accept invitations to join the project or to participate in project feedback studies. You can restore archived projects anytime. However, invitations to join the project or to participate in project feedback studies are not restored when an archived project is restored.
