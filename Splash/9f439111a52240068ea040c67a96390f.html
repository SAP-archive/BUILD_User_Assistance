Archive Projects

As owner of a project you can archive and restore a project on the Project page. It is not possible to modify archived projects. You can restore archived projects anytime.

Prerequisites

You must have created at least one project.

Procedure

  1. On the Project page, select Archive Project.
    The Archive Project dialog is displayed.
  2. On the Archive Project dialog, select Archive.

Results

The project is archived. You can view your archived projects by selecting archived on the Home page.
