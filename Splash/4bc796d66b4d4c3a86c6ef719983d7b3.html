Get Started with Splash

SAP Splash is a UX service that provides design and user experience best practices, tools, and e-learning to organizations looking to implement a design-led and user-centered approach to innovation.

There are a few tasks you need to complete before you can get up and running using Splash. If you want access to all the Splash Features, you have to create an account. It's free to just browse and do a few tasks. The rest requires an account..
