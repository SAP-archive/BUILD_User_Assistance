Upload Files

You can upload files on theFiles tool page. These files can be used in the project when creating prototypes, creating feedback studies, and adding project pictures. You can upload multiple files simultaneously. If you add files to the prototype or to feedback studies in the project they are automatically displayed on the FILES tool page.

Prerequisites

Sign up and sign in on the SPLASH landing page. The Home page is displayed.

Procedure

  1. On the Home page SPLASH landing page, select the thumbnail of the project you want to upload files to.

    The Project page is displayed.

  2. Select the Upload Files thumbnail. If you have previously uploaded files, selct the Tools drop-down list on the Toolbar and select Files.
    The FILES page is displayed.
  3. Select Upload Files and browse for the files to upload, and click Open. You can also upload files by dragging them on to the Files page.

Results

The uploaded files are displayed on the Files page. All documents in the project Files folder are displayed on the Files page if the Everything tab is selected. You can filter the files by selecting the Images, or Documents, Media or Archive tabs.
