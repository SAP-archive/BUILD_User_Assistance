Add Blank Pages

The UI Editor in the Prototype tool allows you to quickly create prototype pages. Add a blank (or freeform) page to create your own unique, customized prototype page.

Prerequisites

Before you can start creating prototype pages, you must first start a new project. See Create a New Project.

Procedure

  1. Open your project.
  2. Select Make a Prototype.
  3. Select the page type Blank and select Add. The page map displays with a thumnbail of the page you just added.
  4. In the page map view, select New Page, then select the page type, to add more pages.
  5. Select a page to open it in the UI Editor.
  6. To add more pages, select +New Page, in the Outline tab on the left panel and select the page type you want to add..
  7. Use the Outline in the content panel on the left work with pages and content::
    • Rename Page: To rename a page, select the page name and type in a new name.
    • Delete Page: To delete a page, select the page name and Delete or Backspace. All content on that page is deleted. To undo a delete, select the undo button on the top toolbar.
    • Add Page: Select +New Page to add more pages to your prototype. Select either Freeform Page or Responsive Page.
  8. Add an image to your page in any of these ways:
    • Drag and drop an image from your computer to the canvas.
    • Drag and drop an uploaded image from the Files tab in the UI Editor to the canvas.
    • Upload an image using the Files tool in the Toolpicker. See Upload Files for details.
  9. To resize the image, select it, and use the Scaling properties on the right panel.
  10. Now you can begin adding UI Controls to make your prototype interactive. See Add UI Controls for more information.
