Add a Project Picture

As project owner you can display a picture on the project tile. Adding a unique picture to each of your project thumbnails makes it easier to identify the project at a glance, and lends your project an impressive visual identity.

Prerequisites

You must sign in to SPLASH in order add a picture to your project thumbnail. You must also have created a project.

Procedure

  1. On the Home page, select the camera icon on thumbnail of a project.
    The Select from files dialog is displayed.
  2. Click Upload From Your Computer to browse for and upload a image. You can also drag images from your computer onto the dialog.
  3. Select the image in Available File section.
    The is selected.
  4. Select Select.
    The image is displayed on the project thumbnail.

Results

The image is displayed on the project tile. All invited members also see the picture.
