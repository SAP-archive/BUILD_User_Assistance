Create a Project by Cloning

You can create new projects by cloining projects created and shared by other SPLASH users. You can do this from your SPLASHHOME page.

Prerequisites

You must sign in to SPLASH in order to create a new project by cloning an existing project.

Procedure

  1. You can clone a project from your SPLASH Home page or directly from your Gallery page. Do one of the following:
    • On the Home page select Gallery in the footer.
    • On the Home page select the Clone An Existing App From The Gallery.
    The SPLASHGallery pages is displayed.
  2. Select the project you want to clone.

    The Gallery Project page for the project is displayed.

  3. Select Clone.

    The project is cloned.

Results

The cloned project and clones of the public files in the project are displayed on your Project page. You can invite team members, create feedback studies, and modify the prototype pages. The following details are not cloned:
  • The date and time that project files were uploaded.
  • The names of the SPLASH users who uploaded files.
  • The names and details of the original SPLASH users who owned the project or were team members on the project.
