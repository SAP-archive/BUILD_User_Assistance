Prototype

The Prototype tab displays the fully-interactive prototype upon which the task was based, and displays participants' responses to each page of the prototype and to the overall task.

Prototype

You can do the following on the Prototype tab:
  • Select the Show Annotations button to do the following:

    • View all annotations icons dropped on the page. The page fades significantly so that you can view the annotation icons more easily.

    • View all the comments entered about this task in the Coments tab on the right-hand panel. In addition, you can filter the comments entered by sentiment.
    • Select the Show Heatmap button to show a heatmap that represents the frequency and location of user-selection on the page of the prototype currently displayed.
    • Interact with the prototype. You can do this by navigating through and using the prototypes itself, or by selecting the Previous or Next button in the right-hand-side panel.
