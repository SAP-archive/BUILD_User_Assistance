Add Objects from Excel

You can add objects or an entire data model from an Excel spreadsheet.

Prerequisites

You or a data modeling specialist have created a data model in Excel and the file is on your local hard drive or network.

Procedure

  1. You can import objects from Excel in one of three ways:
    • Under Data, select the import link.
    • In the Data Editor, select the Import objects from an Excel file link.
    • Select  Add Objects  Manually .

  2. Browse to locate and select the file you want to import.

    The objects and their properties are displayed in the Data Editor. You can then add, delete, or edit objects and their properties as well as add more objects manually or from search.

    Note

    We provide sample Excel files to help you create data models.
