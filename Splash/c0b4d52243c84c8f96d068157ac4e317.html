Add Objects to a Data Model Manually

This section describes how to add objects to a data model manually.

Prerequisites

Access the Data Editor from the Data tab.

Procedure

  1. The first time you access the data editor, you can select the link Create objects manually.

    You may also select  Add Objects  Manually  from the menu.

  2. Type the name of the new object.
    Note

    All changes to your data model are saved automatically. You can drag objects to various locations in the editor.

    After you add data objects to your data model, you must add properties to the objects and create relationships between objects.
