Remove Project Team Members

Prerequisites

Procedure

  1. On the Home page, select the project thumbnail of the project you want remove a team member from.

    The Project page is displayed. An avatar for each project team member with their name below is displayed in the Project Team section.

  2. Select the avatar of the team member you want to remove and select Remove From Team.

Results

The selected team member removed from the project.
