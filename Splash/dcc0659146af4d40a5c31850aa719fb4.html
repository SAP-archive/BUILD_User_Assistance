Controls Tab

The UI Editor includes built-in Open UI5 UI controls you can use to make prototypes interactive without writing any code.

  1. In the UI Editor, enter a control name in the Search box to look for a specific control. You can also go to the Files and Data tabs next to the Controls.

  2. Drag the control you want onto the canvas and apply properties and actions to customize what the control does.

Check out the Open UI5 Developer Guide and the UI5 Controls Reference for more information.
