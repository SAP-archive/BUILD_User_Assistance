Access the Gallery

You can access the Gallery from the SPLASH landing page in three different ways: the Gallery tab, the interactive image, and by using the scroll bar.

Procedure

  • From the Gallery tab:
    • On the SPLASH landing page, select the Gallery tab on the header.
      Note If you are signed in, the header will appear in the footer.
  • The interactive image:
    • The laptop screen features different areas of SPLASH. Select the screen when Explore our gallery appears.

    A carousel of the Gallery is displayed. To go to the Gallery, select View More Projects.

  • Using the scroll bar:
    • Scroll down to the next screen. You can also select the down arrow.

    A carousel of the Gallery is displayed. To go to the Gallery, select View More Projects.
