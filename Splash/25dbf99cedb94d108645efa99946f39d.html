Ask a Question

You can add images to SPLASH feedback studies and ask questions about the images.

Prerequisites

Create a feedback study and enter a name and description. You must also have images that you want to ask questions about.

Procedure

  1. Select the Ask a question tile on the Feedback tool page.

    The Select Images dialog is displayed.

  2. To upload images, select the Upload Images button, and browse to the images you want to include in your study.
    The images are upoaded and displayed on the Select Images dialog.
  3. Select the images you want to include, and select Select.

    A thumbnail of each image is displayed on the Feedback tool page.

  4. Select the thumbnail of the first image to add a question about the image.

    The Create a Question dialog is displayed.

  5. In the right-hand panel, enter a question about the image in the Question text box.
  6. In the right panel, specify how you want participants to respond to the question by selecting one of the following options:
    • Answer through annotation only - specifies that participants respond by selecting on the image to drop Annotation icons onto it, and optionally enter a comment with each icon. You can limit the number of annotations each particpant can drop on the image by selecting the Limit annotations to check box and entering the maximum number of annotations.
    • Answer through free text - specifies that participants can respond by entering an answer. Participants can also voluntarily drop Annotation icons on the image and enter comments for each annotation.
    • Answer through multiple choice - specifies that participants can choose their responses to the question from a number of multiple choices. You must enter a minimum of two choices in the multiple-choice text boxes. You can add additional multiple-choice text boxes by selecting +Add choice . You can select Enable multiple answers to specify that particpants can select more than one choice in reponse to the question. Participants can also voluntarily drop Annotation icons on the image and enter comments for each annotation.
  7. Select if you want to ask another question about this image.
  8. When you have entered your questions about the image, select Done.
    The Feedback page is displayed
  9. Select another image and enter your questions.

Results

The questions are created and an thumbnail image for each question is displayed on the Feedback tool page. You can drag the thumbnails to reorder images and prototypes . Alternatively, you can add another question by selecting the Select Images button, or add a task by selecting the Select a Prototype button.
