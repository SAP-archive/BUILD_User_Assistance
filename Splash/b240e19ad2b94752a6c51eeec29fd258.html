Archiving Studies

Project owners can archive paused studies at anytime. If a study is archived it is not possible for participants provide further feedback to archived studies, and the study-link is not longer available.

Prerequisites

You must create a project before you can archive it.

Procedure

  1. On the Project page, select the project thumbnail that contains the feedback study you want to archive.
    The Feedback results page is displayed.
  2. Select the Settings tab.
  3. Select I want to archive this study.
  4. Select Archive.

Results

The study is archived. Archived studies are displayed from the Archive tab on the Feedback results page, when the arhived studies can be restored.
