Add Objects by Search

You can add objects from business catalogs by using the Search feature.

Prerequisites

Access the Data Editor from the Data tab.

Procedure

  1. The first time you access the data editor, you can select the link Import objects from business catalogs using search.

    You can also select  Add Objects  Manually  from the menu.

  2. In the Search for Data Objects screen, type the name of the object or catalog that you are searching for, then press Enter.

  3. Select one of the objects found in the catalog.
  4. In the next screen, select what you want to add.

    You can add just the object or the object and its relationships. If you select the object and its relationships, you import the object and its children.
