Actions

Actions describe what the UI control does, the actions you want the control to have when the user interacts with the control.

In the UI Editor, when you select a control on your prototype page, the control properties and actions display in the right panel. For example, you can configure a button or hotspot to navigate to page 3 or show an alert or toast message when the user selects it.
