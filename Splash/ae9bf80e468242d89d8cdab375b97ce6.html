Tree View

In the UI Editor, the Tree view is an outline of your prototype contents, including controls, pages, and images.

In the outline view, you see a list of prototype pages, images, controls. In the outline, you can:

  • Rename a Page: Select the page name to highlight it, and enter a new page name.

  • Delete a Page or Control: Select the page name or control and Backspace or Delete.

  • View Page contents: Select the arrow to the left of the page name to expand the view.

Select +NewPage to add pages to your prototype. You can also delete controls in this view.
