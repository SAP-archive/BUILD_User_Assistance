Create a Task

To add a task to a feedback study, you upload an interactive prototype, select Start and Target pages, and enter some guidance for users to follow. You can add as many prototypes in a single study, and set a different task for each. You can set multiple tasks on a single prototype by uploading the same prototype multiple times. For each task study participants will interact with the prototype and, based on the user guidance, select the I Have Finished This Task button when they have displayed the Target page. 

Prerequisites

Create a feedback study and enter a name and description. You must also have a compressed (ZIP) HTML prototype application available to base the task on.

Procedure

  1. On the Feedback tool page, select the Create a task tile.

    The Select a Prototype dialog is displayed.

  2. Select the upload link and browse for the compressed (ZIP) HTML prototype application to base the task, and select Open.

    If you previously uploaded a HTML prototype you want to use select the radio button for the prototype.

    The prototype is uploaded, and available to be used as the basis for the task.

  3. Select the Next button.

    The Create a task dialog is displayed.

  4. Enter a name for the task in the Name the Task text box.
  5. Select on any pages in the prototype that you do not want to include in the task.

    The pages you select on are deselected and not included in the task.

  6. Select the Select button.

    The task is displayed on the Study page.

  7. Select the task thumbnail.

    The Define the Task page is displayed.

  8. Rollover the prototype page you want the the task to begin with, and select Start.

    The page is assigned as the Start page for the task.

  9. Rollover the prototype page you want to make the target for the task, and select Target.

    If you do no select a Target page it will not be possible for participants to fail the task.

    The page is assigned as the Target page for the task.

  10. In the User guidance text box enter a clear description of what the participants of the study are required to do to complete the task.

    For example, you may describe the target page which participants are required to display; or you may describe a specific task that is completed if the target page is displayed.

  11. Select the Save and Close button.

Results

The task is created, and a thumbnail representing it is displayed on the Study page. You can select and drag to reorder images and prototypes. You can select the Publish to publish the study. Alternatively, you can add another task by selecting the New Task button, or add a question by selecting the New Question button.
