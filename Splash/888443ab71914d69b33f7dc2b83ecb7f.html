Create Relationships Between Objects

After you add objects to the data model, you need to create relationships between the objects.

Procedure

  1. In the Data Editor, select the object that you want to create a relationship from.
  2. In the right-side panel, select Relationships, then + New Relationship.
  3. Type a name for the relationhip.
  4. Select a target object from the dropdown list, and then select the cardinality, 1 or n.

    Note

    You can also select the blue circle on the corner of the object and drag a link to the corner of another object to create a relationship between the objects.
