Change Project Owners

The current owner of a project can make any member of a project an owner. Only project owners can archive projects.

Prerequisites

There must me more than one team member in the project.

Procedure

  1. On the Home page, select the project thumbnail of the project you want to change ownership of.

    The PROJECT page is displayed. An avatar for each project team member with their name below is displayed in the Project Team section. Owner is displayed on the owner's avatar.

  2. Select the avatar of the team member you want to make the project owner and select Transfer Proect Ownership.

Results

The selected team member is now the owner of the project.
