Create Feedback Studies

You can create multiple feedback studies in each project. First you add a name and description of the study. You add images to the study, and then add questions about those images for participants to answer. You can add HTML prototypes to the study, and create tasks for participants to complete using the prototypes. Then you publish the study and invite people to take partl.

Prerequisites

You must have created a project or be a member of an existing project in order an create a feedback study.

Procedure

  1. On the Home page, select a project tile.

    The Project page is displayed.

  2. In the Studies section select Create New Study.

    The Study Name and Description dialog is displayed.

  3. Enter a name and description for the study.

    For more information, see Guidelines for Study Names and Descriptions

  4. Select Save.

    The Feedback tool page is displayed. To add content for study participants to engage with, you add questions and tasks to your study.

  5. Create a task or ask a question.
  6. You can select and drag image and task thumbnails on the study page to re-order them.
  7. Select the Preview icon to preview your study.

    You preview your feedback study to display the feedback study on a new tab as participants in the study will see it. You can answer questions about images in a preview but the answers are not saved.

  8. To publish the study, select Publish.

Results

A unique study-link for your feedback study is displayed, and you are able to send the study-link to people that you want to provide feedback..
