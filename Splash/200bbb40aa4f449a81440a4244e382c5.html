Get Started with Admin Console

As a BUILD Administrator, you need to follow these initial steps before users can use the system. You will receive a Welcome email from the SAP BUILD team with instructions on how to get started as BUILD Administrator.

Context

To enable users to use the BUILD system:

Procedure

  1. Open the Welcome email and select Activate Me to create your Cloud Identity.
  2. In the Welcome email, also select the Set things up in the Admin console.

    After you log in with the Cloud Identity, the BUILD Admin console page opens.

  3. In the left sidebar, select Access in the toolbar on the left.
  4. Verify that your domain access is listed and configured.

    Admin access level for your domain is automatically configured.

    Note

    Do not add domains such as gmail.com or yahoo.com because this would allow access to all Gmail or Yahoo users.

  5. Your BUILD users need Full Access instead of Guest With Study or Project Invitation. Change access levels and add new domains or as needed.
  6. Select Users in the toolbar, then select your user name and make sure you are set as a Standard User (if you need this access level).
  7. Send the system URL to your users. After users create their Cloud Identity user, they are automatically BUILD standard users. No additional admin changes are necessary.

Next Steps

The BUILD Admin console allows the BUILD Admin to manage users, domains, and view audit logs. For more information about the BUILD Admin console, see the Admin Console.
