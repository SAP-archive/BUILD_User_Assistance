Smart Template List Report Page

The List Report is the first page of a smart template prototype and is used to filter a few dedicated objects from a large set of objects. The List Report contains a filter bar above a table.

The default list report contains the following:

  • Filter bar with one filter

  • Table

These elements are contained in the Outline and are displayed in the Preview screen.

Filter Bar Controls

You add filters to the List Report by dragging the Filter control to the Outline pane. Add the properties for each filter under Filter on the right side of the screen.

Table Controls

You can add one or more action buttons as well as columns to the table by dragging them to the Outline. You add properties for each action button or column in the on the right side of the screen.

Note

As you add filters, columns, buttons, and properties to your list report, the changes are displayed in the Preview pane. You cannot make changes in the Preview pane, however. You can change the order of the filters and the table columns in the Outline.
