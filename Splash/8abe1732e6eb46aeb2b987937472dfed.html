Project Thumbnail

In the Gallery, each project thumbnail shows you a snapshot of the project. You can see the project's picture, short description, associated tags, and how many likes and views the project has.

On the front of the project thumbnail, the following is displayed:

  • Project title

  • Prototype image

  • Views

  • Likes

Note When you are signed in, the clone icon is visible. You can only view, like, and clone a project if you are signed in.

When you hover your mouse on the project thumbnail, the following is displayed:

  • Project title

  • Author name

  • Associated tags

To view the project details, select the project thumbnail.
