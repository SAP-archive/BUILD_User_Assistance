Files Tab

The Files tab in the UI Editor lists all the images in your project.

Procedure

  1. In the UI Editor, select the Files tab to see a list of images in your project.

  2. Drag an image from the Files list to the canvas to use in your prototype. You can also drag a file from your computer directly to the canvas and it automatically appears in the Files list.
