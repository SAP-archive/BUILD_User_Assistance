Use Images with Smart Templates

You can use images with Smart Template Object Pages to make them more pertinent or attractive.

Rules for Using Images with Smart Template Object Pages
Because you are going to bind the image to a property, you must respect the following rules:
  • You can only use a data model that has been generated from an Excel sheet.
  • The images must be referenced in the data model as follows: assets/image for example, assets/Bob.jpg

  • The images must be uploaded to the application using the Files Tool.

  • When searching for the image, you must use the same syntax as in the Excel sheet: assets/image
  • For image controls, you must bind the image to the object image property.
You have to: 1. Upload your images in FILES 2. You have to create your data model thanks to an Excel import (this is the only way) 3. For the property corresponding to the image you have to declare the following content: assets/image Ex: assets/Bob.jpg Note: 4. For image controls : Source property you have to bind it to the object image property either by a. Selectin the property in the drop down list b. By dropping the object image property on the Image control in the canvas
Note

you can change the image referenced afterwards in the data sample editor or add new items in the Samples
