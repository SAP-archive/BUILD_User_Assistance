Access Topics

Each forum displays a list of topics that SPLASH users create. You can view the details of each topic by selecting a forum from the Community page.

Select one of the following sections from the Community page: All, Tools, Learning, Design Services, Conference Highlights.

In the topic you select, the page displays the following:

  • Topic name

  • Author name

  • Last reply date

  • Number of comments

  • Number of views

Sorting results:

You can sort results by selecting the Sort By menu and choosing the following choices: Date, Comments, Views.

To create a topic, select Create New Topic.

Note

The user needs to sign in or create an account in order to add comments to the topics.
