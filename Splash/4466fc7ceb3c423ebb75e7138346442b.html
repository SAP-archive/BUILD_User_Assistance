Toolbar

The UI Editor toolbar includes various options to customize your prototype and canvas view.

The toolbar includes these options:
Table 1: UI Editor Toolbar

Toolbar Option

Description

Desktop

Show prototype as it displays on a desktop computer.

Tablet

Show prototype as it displays on a tablet.

Phone

Show prototype as it displays on a mobile phone.

Show/Hide Ruler

Use the ruler to help line up objects on the canvas.

Enable/Disable Guides

Guides display as red outlines on selected objects on the canvas so you can easily see them..

Enable/Disable Snap

Snap allows you to easily position and align objects with other objects on the canvas.

Page Map

Displays prototype page thumbnails and lines to show navigation from one page to another.

Magnification

Use the magnification options on the toolbar to adjust the the canvas for best viewing.

Show/Hide Sidebar (left and right)

Hide the UI Editor side panels for more room viewing the canvas.

Undo/Redo

Undo and Redo actions in the UI Editor.

Create Study

Create a user study to test your prototype on users or colleagues. Get feedback to fine-tune your prototype.

Publish

Create a link to share your prototype with others.

Preview

Preview prototype pages and interactions on each page.
