Access the Data Editor

This topic describes how to access the Data Editor.

Context

You need to create a project before accessing the Data Editor.

Procedure

  1. Do one of the following:
    • If you are creating a new project, select the Page Map, then select Add on the Blank Page tile.
    • If you already have a project, go directly to the first page of your project.
  2. In the next screen, select Blank Page
  3. The next screen in the application displays three tabs. Select the Data tab.

  4. Under the Data tab, select Data Editor.
