Run Prototype in Web IDE

To use Web IDE with BUILD prototypes, you must first set run configurations in Web IDE.

Context

To set run configurations:

Procedure

  1. In Web IDE, select  Run  Run Configurations  and select .
  2. Select New Run Configuration and create a select type Web Application.
  3. Uncheck the Open with Frame option and Save.
    Note

    You can now preview prototype applications after authentication through Cloud ID services, because this launches the application without a test frame.
