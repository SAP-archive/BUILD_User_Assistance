Example: Adding Properties to an Object

In this example, you add properties to the object that you have just created.

Context

We'll add two properties, SupplierName and EmailAddress to the Supplier object of the data model.

Procedure

  1. Select the object that you want to add a property to, in this case, the Product object, then select the Properties tab.
  2. Select New Property below the list of properties.
  3. Type a name for the property, SupplierName and select the type, String.
  4. Add another property, EmailAddress and leave the type as String.

    The new properties appear the list of properties of your object, Supplier.
