Add Objects to a Data Model Manually

This section describes how to add objects to a data model manually.

Context

You are in the Data Editor.

Procedure

  1. Select the Add Objects icon, then select Manually from the menu.

  2. Type the name of the new object.
    Note

    All changes to your data model are saved automatically. You can drag objects to various locations in the page.

Next Steps

After you add data objects to your data model, you must add properties to the objects and create relations between objects.
