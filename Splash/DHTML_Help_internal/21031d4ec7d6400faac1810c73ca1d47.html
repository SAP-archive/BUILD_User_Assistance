Page Map

View the Page Map from the UI Editor or from the Project page. The Page Map displays a map view for your project. Page thumbnails and navigation are also represented in the Page Map.

Context

To view the Page Map:

Procedure

  1. Select one of these methods:
    • Select the Page Map icon from the UI Editor toolbar.
    • Select the Page Map icon on the Project Page.
  2. In the Page Map:
    • View the Home page, indicated by the icon.
    • View thumbnails of all pages in your project.
    • View page navigation (green and gray arrows) in your project that are defined in the UI control properties.
    • Add (New Page) and delete pages (Delete key)
      Figure 1: Prototype Pages Map View
  3. Select a page thumbnail to open it in the UI Editor.
