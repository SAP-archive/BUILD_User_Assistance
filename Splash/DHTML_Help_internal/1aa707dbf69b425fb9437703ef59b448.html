Manage Users

BUILD Administrators can use the Admin Console to view and delete users, and to assign roles.

Users and Roles
To view users:
  1. On the left sidebar, select Users. BUILD users are displayed.
    Change the user view using icons on the top right side:
    • Select to view user summaries.
    • Select to view user details.
  2. To search for a specific user, enter the user name or email address in the search box at the top.
  3. Select a user to view their role.
  4. Check the appropriate user role and select OK.
    • Standard: BUILD user can create and manage projects.
    • Guest: BUILD user can only participate in user studies.
Note If you are a BUILD Administrator, and you want to access BUILD, you need to assign yourself the Standard user role.
Note When you change a user's role from Standard to Guest user OR delete a user, the system automatically deletes the user from all project teams and projects the user owns, and studies and study feedback associated with deleted projects.
Delete Users
  1. To delete a BUILD user, select the icon next to the user (summary or detail view) to delete a BUILD user.
  2. Confirm you want to delete this user and all project-related access, then select Delete.
