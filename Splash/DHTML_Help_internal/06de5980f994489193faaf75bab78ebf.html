Configure the List Report

This section describes how to add and configure elements for the list report.

Context

After you have selected the main object of your prototype, you can configure the list report.

Procedure

  1. In the Outline, select Filter.
  2. Under Properties, type a description for the filter, for example, Name. The description will appear above the filter box in the list report.
  3. In the Filter on field, from the list, select the property on which you want to filter. The properties are those of the main object.
    Note

    You must be in the Outline pane to be able to enter values for the properties.

    Example of a filter, Product Name, bound to the Name property of the Product object.

  4. Select Table Objects in the Outline and type a title for the table in the Properties pane.

    Example Table title

  5. For each column in the table:
    • Select the column in the Outline.
    • Under Properties, type a title for the column and select the value or values that you would like to have appear in the column. For example, you can have Price and Currency appear in the same column.

  6. If you want to add columns or action buttons to the table, under Table Controls, select the control and drag it to the Table Products in the Outline.

    At any time, you can select the Preview button to preview your list report.

  7. When you are ready to populate your list report, select Preview and then the Go button.
