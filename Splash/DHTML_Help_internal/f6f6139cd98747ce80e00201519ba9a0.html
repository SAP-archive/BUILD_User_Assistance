Add a Second Object Page

Once you have configured the first level of object pages, you can continue to a lower level of the data model.

Procedure

  1. Select  +New Page  Object Page Smart Template .
  2. Under Properties select the Main Object of your object page from the list of available objects.

    When you select the main object you bind the object to the object page.

  3. As you did in the first object page, configure the header and the first section. Depending on the data in your model, you may not need the table section and you may delete it.
  4. Publish the prototype. Now you can navigate to the second level of the object pages.
