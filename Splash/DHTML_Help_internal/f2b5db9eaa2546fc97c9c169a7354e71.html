Add a List Report

Once you have created your project, you can start using the smart templates. Start by adding a list report, and then upload a data model.

Procedure

  1. Select your project, then select Add on the List Report tile on the next screen.

  2. Select Page 1. The UI Editor for Smart Templates opens.

    Page 1 is displayed in the Outline pane with a filter bar and table objects.

  3. Select the Data tab on the left side of the editor.

  4. Select the Data Editor button.

    The Data Editor opens. This is where you'll upload the Excel file that will generate the data model.
  5. Select the Add Objects button, then From Excel.

    While BUILD allows you to add objects manually or from business catalogs, most data models are generated from Excel files.

  6. Browse to locate the file that you want to upload. When the file is uploaded, BUILD generates the data model.

  7. Select the Back button to return to the list report. The List Report section under Properties is pre-populated with properties from your data model. Select the object that you want to use as your Main Object. Each list report and object page has its main object. When you select the main object you bind the list report to the object.

    Once you have selected your main object, you are ready to configure your list report.
