Configure the Smart Template List Report

This section describes how to add and configure elements for the list report.

Prerequisites

You must have selected a main object of your prototype before you can configure the list report.

Procedure

  1. By default, the application allows free search in the filter bar. If you do not want to allow free search, select the Filter Bar control in the Outline, then change the setting in the right side of the canvas under Filter Bar.

  2. In the Outline, select Filter.
  3. On the right side of the screen, enter a description for the filter, for example, Name. The description will appear above the filter box in the list report.
  4. In the Filter on field, select the property that you want to filter from the list. The properties are those of the main object.
    Note You can add controls such as filters, by dragging the control to the Outline. A dotted green box shows you where to drop the control.

    Example of two filters, Burger Name, which is bound to the Name property of the Burger object, and Price, which is bound to the Price property of the Burger object.

  5. Select Table Objects in the Outline and enter a title for the table in the Title box on the right side of the screen.

    Example of the table title.

  6. For each column in the table:
    • Select the column in the Outline.
    • Enter a title for the column in the Column Header field, and select the value or values that you would like to have appear in the column. For example, you can have Price and Currency appear in the same column. The Boldface Value appears in boldface type in the table. You can also choose to link to a URL from the column, as well as set the responsiveness of the column.

  7. If you want to add columns or action buttons to the table, under Table Controls, select the control and drag it to the Table in the Outline.

    At any time, you can select the Preview button to preview your list report.

  8. When you are ready to populate your list report, select Preview and then the Go button.

    Note

    If you have finished configuring the filter bar and the table, you are ready to configure the Smart Template Object Page.
