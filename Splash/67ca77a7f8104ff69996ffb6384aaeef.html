Add and Configure a Smart Template Object Page

The object page of a smart template contains information about a specific object.

Prerequisites

You must have configured a Smart Template List Report.

Procedure

  1. In the Outline of the Smart Template List Report, select the table, then, on the right side of the screen, under Table, select +New Page> Smart Template Object Page.

    The object page, Page 2, opens. By default, the object page contains the following:

    • Header

    • Form section

    • Table section

  2. Select Page 2 in the Outline. The Main Object is selected by default. It is the same as the main object of the list report.
  3. In the Outline select Header, then on the right side of the screen, under Header, configure the title and the description of the object page:
    • Enter a Header bar text, for example, Burgers.
    • Select a Title, for example, the Name property.
    • Select a Description, for example, the Description property.
    • If you have already uploaded images to the application using the Tool PickerTool Picker, you can choose an image to display in the header from the list under the Files tab.

    You can also enter a custom value.

    Note

    You can add controls to the sections in your object page by dragging the controls to the Outline. A dotted green box shows you where you can drop the control.

  4. If you want to add a KPI:
    • Select KPI in the Outline.
    • On the right side of the screen under KPI, enter a title for the KPI..
    • Select a value and a currency from the dropdown lists where appropriate.

    You can have more than one KPI in your object page. If you don't want to use any KPIs, you can delete the default KPI in the Outline.

  5. If you want to add an action button to the header, select the Action Button control, drag it to the header, then add text on the right side of the page.
  6. Configure the first section of the object page by expanding the section in the outline and configuring the form elements on the right side of the page.
    Form Control Values
    Form Group Title (not mandatory)
    Label Label (name) - Value (may be custom value)

    You can have more than one value for each label such as Currency. You can also set values for the following:

    • URL: URL to which you can navigate from the Label provided that the objects have a 1 to 1 relationship.

    • Mandatory
    • Read-only: when the form is in edit mode, the mandatory fields are designated with an asterisk
    • Edit Mode: describes the way a smart template object page can be edited. Input can be: Standard Input, Dropdown list, or a Value Help Dialog box (F4). The Edit link is in the upper-right corner of the Smart Template Object Page.

    Note

    You can have multiple form and table sections in a Smart Template Object Page.

  7. Expand the second section and select Table in the Outline.
  8. Configure the elements of the table on the right side of the screen. Select the Items that you want displayed in the rows of the table
    Table Control Values
    Table Items: what you want displayed in the table rows.
    Column Header - at least one value.
    Note

    An action is selected by default to allow you to navigate to the next object page in the prototype from the table.

  9. Select the Preview button to view your object page with its values.

    Note

    This object page contains the following elements:

    • Header bar: Burgers
    • Header title: Burger name, Blue Burger
    • Description: Description of the burger
    • Image of the burger
    • One section: no name
    • One form group in the section: no name
    • Two form elements in the form group: Labels Price and ID
    • A second section: Details
    • A table in the second section with three columns that display the ingredient ID, the Name, and Category

    If you want to continue adding or editing data in the object page, select Editor. As with all forms, there is no need to save your work. The application automatically saves it for you.
