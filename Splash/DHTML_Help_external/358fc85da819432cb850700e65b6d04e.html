Restarting Studies

Project owners can restart paused or archived studies at anytime.

Procedure

  1. Select the project tile that contains the user research study on the Home page.
  2. Select Research on the navigation bar.
  3. Select Settings.
  4. Select I want to archive this study .
  5. Select RESTART.

Results

The study is now active again. Active studies are displayed from the Active tab on the Research page.
