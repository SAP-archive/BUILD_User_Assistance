Create BUILD User

The first time you access BUILD you need to sign up. It's free and easy to do. User access is determined by your BUILD administrator. You may have limited access to some features until your admin sets permissions for you.

  1. Select Join on the BUILD Sign In page.
  2. Enter a name that you want your team members to see. This is your screen name.
  3. Enter a valid email address. BUILD needs to verify your email.
  4. Enter a password to access BUILD. You can change your password later. Your password must be 6 to 40 characters long and include any combination of 3 of the following:
    • Uppercase letters
    • Lowercase letters
    • Numbers
  5. Select Join. You can sign in and start using BUILD.
  6. Check your email for a Welcome to BUILD message. If you do not receive one, check your spam folder for an email from do.not.reply@sap.com. If you still don't receive the email, check under Contact Info on the Profile page after you sign in and select Resend verification email.
  7. Open the email and select the link to verify your email address. Your account is now activated.
  8. Sign in and start using BUILD.
    Note

    The Getting Started page displays when you first sign in and includes videos to get you quickly started using BUILD. If you don't want this page displayed next time you sign in, select the Don't show this again check box at the bottom of the Getting Started page. You can still access videos and help using the help button in BUILD.
