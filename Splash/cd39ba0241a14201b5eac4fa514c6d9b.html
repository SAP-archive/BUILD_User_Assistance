Create a New Project

You create a new projects from the Home page displayed when you sign in on the SPLASH landing page.

Prerequisites

You must sign in to SPLASH in order to new create projects.

Procedure

  1. Select Create New Project on the Home page.

    The Project Name and Description text-entry boxes are displayed.

  2. Enter a name and a description for the project, and click Create Project.

    The project is created.

Results

You can access the project anytime by signing in to SPLASH and selecting the thumbnail for the project on Home page.
