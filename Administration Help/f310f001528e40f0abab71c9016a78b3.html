Set Domain Access

BUILD Admin can add domains and assign roles and permissions to each domain. Then users from these domains can be assigned roles and access BUILD.

Context

To add a new domain:

Procedure

  1. Select Access in the left sidebar. The Access List is displayed. Default access level should be set to Full Access
  2. Select New Domain to add a new domain to BUILD and assign permissions.
  3. Enter the domain name in the format <domainname>domain.com, for example sap.com.
  4. Select the BUILD access level from the dropdown menu. You can choose from the following access levels:

    Access Level

    Permissions

    No Access

    Users cannot access BUILD at all.

    Study by Invitation Only

    Users can access a BUILD study by invitation only as a Guest user. URL in invitation provides users access to study.

    Study or Project by Invitation Only

    Users can access a BUILD study OR project by invitation only. URL in invitation provides users access to study or project.

    Guest Can Access Study by Invitation

    Users can sign in to BUILD as a Guest and access any Study by invitation. URL in invitation provides users access to study.Users can share Study URL with other users.

    Guest Can Access Study or Project by Invitation

    User can sign in to BUILD as a Guest and access any Study. Users can access Project by invitation only. Users can share Study URL with others, but cannot share Project URL.

    Full Access

    User has full access to all BUILD features
  5. Repeat these steps for all the domains you want to add.
